## ğŸ“– About The Project

**Stellar Yield** is a high-performance web application designed for real estate investors who demand speed, accuracy, and privacy. Unlike cluttered, ad-heavy calculators, Stellar Yield offers a sleek, galaxy-inspired interface that allows you to calculate **Gross Yield**, **Net Yield**, and **Annual Cash Flow** in milliseconds.

Built with modern web technologies, it features a comprehensive educational hub (SEO Article), compliance pages (GDPR/CCPA/DMCA), and engagement tools like result sharing and printing.

## âœ¨ Key Features

*   **Real-Time Financial Analysis**: Instantly computes yields and cash flow as you type.
*   **Immersive Galaxy UI**: A deep-space aesthetic with subtle animations for a premium user experience.
*   **Privacy-First Design**: 100% Client-side execution. No data is sent to servers; your financial details remain in your browser.
*   **Engagement Tools**: Built-in "Share Result" and "Print Report" functionality.
*   **SEO Optimized**: Fully integrated JSON-LD Schema, meta tags, and high-quality content structure.

## ğŸš€ Getting Started

To run this project locally:

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/hsinidev/stellar-yield-calculator.git
    ```
2.  **Navigate to the folder:**
    ```bash
    cd stellar-yield-calculator
    ```
3.  **Serve the files:**
    You need a simple HTTP server because of ES Module imports.
    *   **Python:** `python -m http.server`
    *   **Node (serve):** `npx serve`
    *   **VS Code:** Use the "Live Server" extension.

4.  **Open in Browser:** Go to `http://localhost:8000` (or port specified).
